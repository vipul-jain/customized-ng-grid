Rich Grid Control [AngularJS Ng-Grid]

When you use Angular Js Ng-Grid with pagination you will notice that all sort of actions like sorting, filtering, selection do not remain persistent; i.e they are relevant to the present page (page here refers to the pagination) only. whenever you move on to next page[pagination] the actions[sorting/filtering/selections] are lost. Here is the fix to all those issues by creating Templates and manually handling the functions for sorting , selecting and filtering.

Rich Grid features

  • Summarized data information
  • Data is been loaded from json file
  • Pagination as per choice [you can vary page size as per your need]
  • Sorting on whole Data Ascending/Descending [clicking on the column Label will perform sorting]
  • Filtering each column [enter few characters in column header textBox to filter data ]
  • Column selection as per need [notice the small circle with inverted triangle in the top-right hand side]
  • Inline Editing [Name/City/Email are editable just double click on any cell to update it]
  • Selecting rows [click on the checkboxes to select rows which will be displayed in below grid]
